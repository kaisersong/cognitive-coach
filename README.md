# cognitive-coach

`cognitive-coach` 是一个认知元能力教练型 skill，用来通过苏格拉底式提问帮助用户把问题想深、想透，而不是直接给答案。

## 状态

**未发布。当前仅供作者自己测试、验证和迭代使用。**

这意味着：

- 目前不是稳定对外版本
- 提示词、诊断流程和对话节奏仍可能继续调整
- 暂未承诺兼容性、版本语义或公开分发方式

## 当前内容

- `SKILL.md`
  - skill 定义与完整对话工作流

## 设计目标

- 用 9 种认知元能力框架诊断思维盲点
- 通过单轮单问的苏格拉底式提问推进对话
- 支持两种入口：
  - 用户带着已有分析来诊断盲点
  - 用户从零开始梳理一个复杂问题

## 使用边界

- 这是一个思维教练，不是答案生成器
- 核心交付是问题，不是结论
- 优先帮助用户发现自己的认知缺口、权重冲突和隐藏假设
